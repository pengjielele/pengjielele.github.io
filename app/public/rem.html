Adjust size of secondary elements to be small, medium or large

This page contains a variety of interface elements. They are not intended to actually work, and instead are here to show how a subset of elements with dimensions specified in rem can change size when <html> element font size is adjusted. Meanwhile, all other content remains unaffected.

Since this is a technical demo, things will not look pretty, and that's OK.
